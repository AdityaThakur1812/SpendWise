# SpendWise

A modern expense tracker built using HTML, CSS and JavaScript.

## Features

- Add Transactions
- Delete Transactions
- Income & Expense Tracking
- Category Management
- Local Storage Persistence
- Responsive UI
- Modern Dashboard

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Future Improvements

- Search
- Filters
- Charts
- Dark Mode
- Backend Integration
